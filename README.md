# 🎵 MelodiKit

A modern music playlist management web application built with **Spring Boot**, **Thymeleaf**, **Spring Data JPA**, and **H2 Database**.

## 📖 About

MelodiKit is a web application that allows users to manage songs, genres, and playlists through a clean and user-friendly interface.

The project demonstrates the fundamentals of backend web development using the Spring ecosystem and follows the MVC (Model–View–Controller) architecture.

Developed as part of the **Internet and Web Programming** course.

---

## ✨ Features

- 🎵 Manage Songs (Create, Read, Update, Delete)
- 📂 Manage Music Genres
- 🎼 Create and Organize Playlists
- 🔍 Search and Filter Songs
- 💾 H2 Database Integration
- 🖥️ Responsive Interface with Thymeleaf
- ⚡ Spring Boot MVC Architecture

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/burakaydins/melodikit.git
```

Navigate into the project:

```bash
cd melodikit
```

Run the application:

```bash
./mvnw spring-boot:run
```

Open your browser:

```
http://localhost:8080
```
